# Lab Inventory Management

✨ https://lab-inventory-mgmt.onrender.com ✨

Simple web app with an analytics dashboard that allows you to add, delete, and modify medicines. 
- Inspired by [CrelioHealth](https://creliohealth.com/lab-software/inventory-management/)
- Uses React & Axios, Node.js & Express, Sequelize & Postgres, and this [bootstrap template](https://adminkit.io). Hosted at [Render.com](https://render.com)
